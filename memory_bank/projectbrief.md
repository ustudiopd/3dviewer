# 프로젝트 개요 (Project Brief)

## 1. 프로젝트 목표  
- **핵심 문제**: 3D 모델(GLTF/GLB)을 웹 브라우저에서 쉽게 공유하고 시각화할 수 있는 플랫폼 제공
- **최종 목표**: 
  - 관리자가 3D 모델을 업로드하고 데모를 생성
  - 고유한 접근 코드를 통해 3D 모델을 안전하게 공유
  - 웹 브라우저에서 별도 소프트웨어 없이 3D 모델을 인터랙티브하게 조작 가능
- **기대 효과**: 
  - 3D 모델 공유 및 프레젠테이션 프로세스 간소화
  - 접근 로그를 통한 사용자 분석 가능
  - 반응형 디자인으로 다양한 디바이스에서 접근 가능

## 2. 핵심 기능  
- **3D 모델 뷰어**: 
  - `@google/model-viewer` 기반 웹 기반 3D 모델 렌더링
  - 드래그로 회전, 휠로 확대/축소, 우클릭으로 이동
  - Babylon.js Sandbox 스타일 그라데이션 배경
  - HDR 환경 맵을 통한 고품질 조명 및 바닥 반사 효과
  - 로딩 진행률 표시
- **관리자 대시보드**: 
  - 3D 모델 업로드 및 관리 (GLB/GLTF, Draco 압축, KTX2 텍스처 지원)
  - 데모 생성 및 관리 (고유 접근 코드 자동 생성)
  - 접속 로그 조회 및 통계
  - 모델 및 데모 메모 기능
- **인증 시스템**: 
  - 관리자 로그인 (비밀번호 기반)
  - 세션 관리
- **접근 로그**: 
  - 데모별 접속 기록 수집
  - 사용자 에이전트 정보 저장

## 3. 타겟 사용자  
- **주요 사용자**: 
  - 3D 디자이너 및 개발자 (모델 업로드 및 공유)
  - 클라이언트 및 협업 파트너 (3D 모델 시각화 및 리뷰)
  - 마케팅 및 프레젠테이션 담당자 (3D 제품 시연)
